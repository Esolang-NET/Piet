# PT0007: Required output interface not provided

## Cause
The Piet program requires output but the target method has no supported output destination (e.g., `TextWriter` or `PipeWriter` parameter).

## Solution
Add a `TextWriter` or `PipeWriter` parameter to the method signature.

## Example

```cs
partial class Sample
{
    // Incorrect: Piet program requires output but no output parameter is provided
    [GeneratePietMethod("output_program.png")]
    public static partial void Invalid();
}
```
