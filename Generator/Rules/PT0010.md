# PT0010: Input interface provided but not required

## Cause
An input parameter is provided but the Piet program does not require input.

## Solution
Remove the unused input parameter.

## Example

```cs
partial class Sample
{
    // Incorrect: Piet program does not require input
    [GeneratePietMethod("no_input_program.png")]
    public static partial void Invalid(System.IO.TextReader input);
}
```
