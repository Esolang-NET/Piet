# PT0001: Invalid image path parameter

## Cause
The `imagePath` provided to the `GeneratePietMethodAttribute` is null or empty.

## Solution
Ensure a valid, non-empty image path is provided to the attribute.

## Example

```cs
partial class Sample
{
    // Incorrect: Missing or empty image path
    [GeneratePietMethod("")]
    public static partial void Invalid();
}
```
