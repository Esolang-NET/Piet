# PT0005: Image file not found

## Cause
The Piet image file path provided to the `GeneratePietMethodAttribute` cannot be resolved.

## Solution
Ensure the image file exists and is correctly referenced in the project's `AdditionalFiles`.

## Example

```cs
partial class Sample
{
    // Incorrect: "nonexistent.png" is not found
    [GeneratePietMethod("nonexistent.png")]
    public static partial void Invalid();
}
```
