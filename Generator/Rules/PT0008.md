# PT0008: Required input interface not provided

## Cause
The Piet program requires input but the target method has no supported input source.

## Solution
Add a `TextReader` or `PipeReader` parameter to the method signature.

## Example

```cs
partial class Sample
{
    // Incorrect: Piet program requires input but no input parameter is provided
    [GeneratePietMethod("input_program.png")]
    public static partial void Invalid();
}
```
