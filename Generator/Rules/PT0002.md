# PT0002: Unsupported return type

## Cause
The method's return type is not supported by the Piet source generator.

## Solution
Change the return type to one of the supported types (e.g., `void`, `int`, `string`, `Task`, `ValueTask`, `IEnumerable<byte>`).

## Example

```cs
partial class Sample
{
    // Incorrect: double is not a supported return type
    [GeneratePietMethod("program.png")]
    public static partial double Invalid();
}
```
