# PT0003: Unsupported parameter type

## Cause
The method contains a parameter of an unsupported type.

## Solution
Ensure all parameters are of a supported type (e.g., `string`, `TextReader`, `PipeReader`, `TextWriter`, `PipeWriter`, `CancellationToken`, `ILogger`).

## Example

```cs
partial class Sample
{
    // Incorrect: int is not a supported parameter type
    [GeneratePietMethod("program.png")]
    public static partial void Invalid(int param);
}
```
