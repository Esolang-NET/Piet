# PT0004: Duplicate parameter type

## Cause
The method contains multiple parameters of the same type or conflicting types.

## Solution
Ensure each supported parameter type is only used once per method.

## Example

```cs
partial class Sample
{
    // Incorrect: duplicate CancellationToken parameter
    [GeneratePietMethod("program.png")]
    public static partial void Invalid(System.Threading.CancellationToken ct1, System.Threading.CancellationToken ct2);
}
```
