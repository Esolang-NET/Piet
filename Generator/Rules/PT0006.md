# PT0006: Invalid image format

## Cause
The referenced file is not a supported image format (e.g., PNG, GIF, PPM).

## Solution
Use a valid, supported image file.

## Example

```cs
partial class Sample
{
    // Incorrect: "text.txt" is not a valid image format
    [GeneratePietMethod("text.txt")]
    public static partial void Invalid();
}
```
